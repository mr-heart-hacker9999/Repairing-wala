How to Check if a Component is Damaged:

  • Check for burns, cracks, or discoloration.
  • Use a multimeter to test voltage or resistance.
  • Compare values with datasheet or standard charts.
  • For capacitors, test ESR using ESR meter.
  • Test diodes/transistors with continuity function.

Repair Tips & Tricks

Always discharge capacitors before testing.
Use flux while soldering for better joints.
Never test a circuit while plugged in.
Check fuses before anything else.
Document before dismantling appliances.
